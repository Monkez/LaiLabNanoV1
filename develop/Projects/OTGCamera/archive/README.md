# Archived files

Thư mục này chứa code và tài liệu không tham gia vào bản build chính hiện tại.
Các file được giữ lại tạm thời để tham khảo và có thể khôi phục khi cần.

- `legacy_sources/`: các phiên bản C++ cũ/backup trước `main.cpp` hiện tại.
- `experiments/`: demo IPC, GPIO, Processing visualizer và script kiểm thử phần cứng.
- `legacy_deployment/`: cấu hình triển khai cũ hoặc không phù hợp với BusyBox/SysVinit hiện tại.
- `research_notes/`: tài liệu phân tích/thử nghiệm cũ.
- `references/`: tài liệu tham khảo chưa dùng trực tiếp.

Các thành phần build/runtime chính vẫn nằm ở thư mục gốc: `main.cpp`,
`reset_btn.cpp`, `CMakeLists.txt`, `S98reset_btn`, `S99yolo_camera`, `libs/`,
`python_client/`, `docs/` và các model `.cvimodel`.
